# Code of Conduct

## Our Standards

`unimpeded` is a scientific software project, and we expect all contributors and users to engage professionally and respectfully.

We value:
- Constructive scientific discussion and feedback
- Respectful communication in issues and pull requests
- Collaborative problem-solving
- Helping others learn and improve

We do not tolerate:
- Harassment or discriminatory behavior
- Disrespectful or unprofessional communication
- Personal attacks or inflammatory comments

## Reporting

If you experience or witness unacceptable behavior, please contact [Dily Ong](mailto:dlo26@cam.ac.uk) or [Will Handley](mailto:wh260@cam.ac.uk).

## Scope

This applies to all project spaces including GitHub issues, pull requests, and community discussions.
